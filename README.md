# Гороскоп на сегодня — круговая версия

Статический сайт для размещения на GitHub Pages. Интерфейс рассчитан на один экран: знаки зодиака расположены по кругу, а прогноз выбранного знака выводится в компактной карточке без необходимости прокрутки.

## Что внутри

- `index.html` — основная страница.
- `styles.css` — круговой адаптивный дизайн под экраны iOS, Android, планшеты и ПК.
- `app.js` — логика выбора знака, отображения прогноза и событий аналитики.
- `forecasts-data.js` — локальная база прогнозов: 365 дней × 12 знаков = 4 380 вариантов.
- `analytics-config.js` — настройки аналитики.
- `analytics.js` — подключение Яндекс Метрики / GA4 / Umami.
- `manifest.webmanifest`, `sw.js`, `icons/` — вспомогательные файлы веб-приложения.

## Аналитика

В проекте уже включена Яндекс Метрика:

```js
counterId: 110547743
```

События, которые отправляются в Метрику:

- `forecast_view` — просмотр прогноза;
- `sign_select` — выбор знака;
- `forecast_refresh` — выбор другого варианта прогноза;
- `share` — нажатие кнопки «Поделиться»;
- `fullscreen_open` — запуск во весь экран.

## Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы из архива в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. Выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/root`.
6. Откройте опубликованную ссылку GitHub Pages.

## Изменение счетчика Метрики

Откройте `analytics-config.js` и замените значение:

```js
counterId: 110547743
```

на свой номер счетчика.

## Принцип работы прогнозов

Сайт не использует платные API и серверную часть. Прогноз выбирается локально по текущей дате устройства и выбранному знаку зодиака из файла `forecasts-data.js`. В базе заранее подготовлено 4 380 вариантов.
